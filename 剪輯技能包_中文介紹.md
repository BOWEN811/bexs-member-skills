## 1. `video-shorts`：長片自動切短影音

新增日期：**2026-06-26**

### 這個幫你做什麼

把一支長影片自動讀完，抓出幾段可以發 IG Reels / TikTok / YouTube Shorts 的精華，並輸出多支直式短影音。

適合：

- 交易教學長片
- 盤面講解
- 螢幕錄影
- 有主畫面的知識型影片
- 想一次切 3 支、5 支、8 支短影音

### 你可以這樣講

```text
這支長片幫我切 5 支短影音
```

```text
幫我看這支影片有沒有可以發 Reels 的片段
```

```text
把這支 YouTube 教學切成 Shorts，抓重點就好
```

### 不要拿它做什麼

不要拿它做「純訪談黑底字幕片」。  
如果沒有露臉、沒有主畫面、重點是主持人與受訪者的對話，要用 `interview-reels-dialogue-editor`。

## 2. `palmier-pro-shorts-finisher`：Palmier Pro 短影音精修

新增日期：**2026-06-26**

### 這個幫你做什麼

接續已經存在的 Palmier Pro 專案，幫短影音做視覺精修。

它不是從零開始剪片，而是把已經剪好的短片變得更像可發布成品。

適合：

- 上方加 YouTube 封面感橫幅
- 下方字幕區加 HUD / 交易介面感
- 加黑金紅金融風格包裝
- 加低調水印
- 加片尾金句動畫
- 解決 Palmier 沒刷新、Rick 說「我沒看到」的問題

### 你可以這樣講

```text
這個 Palmier 專案幫我精修，下方字幕區太單調
```

```text
上方幫我加一個 YouTube 封面感的橫幅
```

```text
幫我加黑金紅 HUD，水印要淺一點
```

```text
Palmier 我沒看到更新，你幫我另開副本
```

### 不要拿它做什麼

不要拿它從長片自動切短影音。那是 `video-shorts`。  
不要拿它做純訪談黑底逐字稿。那是 `interview-reels-dialogue-editor`。

---

## 3. `interview-reels-dialogue-editor`：純訪談黑底字幕 Reels

新增日期：**2026-06-26**

### 這個幫你做什麼

把沒有露臉、沒有可用畫面的訪談，做成 9:16 高級黑底逐字稿短片。

這個 skill 是從「粥粥訪談」那次踩過的錯誤固化出來的：  
不能漏字幕、不能混 Rick 與受訪者的聲音、不能先亂塞 B-roll、不能讓 Palmier 快取害人看到舊版。

適合：

- Podcast 感訪談
- 純聲音訪談
- 沒有人物畫面
- 主持人與受訪者要分色
- 每句話都要有字幕
- 黑底、PingFang TC、高級訪談視覺
- IG Reels 安全區排版
- Palmier Pro 裡要看得到剪輯過的 V1/A1 軌道

### 你可以這樣講

```text
這支是純訪談，照粥粥那種黑底高級字幕做
```

```text
這支沒有露臉，幫我做 Podcast 感 Reels
```

```text
主持人跟受訪者要分開，字幕不能漏
```

```text
讀完字幕後再剪，不要先做畫面
```

```text
用 PingFang TC，全部放在 IG 安全區
```

### 這個 skill 的硬規則

- 有人講話就要有字幕。
- Rick 的話不能跑到受訪者名下。
- 受訪者的話不能跑到 Rick 名下。
- 不要只靠聲波猜 speaker，要做人工驗證。
- Review UI 要能 autosave、自動播放下一段、不要每次點完跳回頁首。
- 純訪談的主角是文字、聲音、節奏，不是亂塞素材。

## 4. `rick-yt-publish`：影片上架包

新增日期：**2026-06-26**

### 這個幫你做什麼

剪完影片之後，幫你整理 YouTube / Reels 發布需要的文字素材。

適合：

- YouTube 標題
- 影片簡介
- 章節
- hashtag
- 封面標題
- 封面設計提示詞
- 字幕繁體化與錯字校正

### 你可以這樣講

```text
這支影片給我標題、內文、封面提示詞
```

```text
這支做 YouTube 上架包
```

```text
幫我想更聳動但高級的封面標題
```

```text
我要丟網頁版 GPT 做封面，給我完整提示詞
```

### 不要拿它做什麼

它不是剪輯工具。  
短片本體要依照情境選 `video-shorts`、`palmier-pro-shorts-finisher` 或 `interview-reels-dialogue-editor`。

---

## 怎麼選：三秒判斷法

問自己三個問題：

### 問題 1：我現在有的是長片，還是已剪好的 Palmier 專案？

- 長片，要自動抓精華 → `video-shorts`
- 已經有 Palmier 專案，要變高級 → `palmier-pro-shorts-finisher`

### 問題 2：這支是不是純訪談、沒有露臉、沒有主畫面？

- 是 → `interview-reels-dialogue-editor`
- 不是，有盤面/講師/螢幕畫面 → 通常先用 `video-shorts`

### 問題 3：影片已經完成，只缺發布文字？

- 是 → `rick-yt-publish`

---

## 常見組合流程

### 流程 A：長教學影片變多支 Shorts

1. 用 `video-shorts` 抓出 3-5 支短片。
2. 如果某支畫面太普通，用 `palmier-pro-shorts-finisher` 做視覺精修。
3. 最後用 `rick-yt-publish` 做標題、簡介、封面文案。

### 流程 B：純訪談變 IG Reels

1. 用 `interview-reels-dialogue-editor` 讀完字幕與說話者。
2. 做 speaker review，確認主持人/受訪者沒有混。
3. 產出黑底高級字幕版。
4. 用 `rick-yt-publish` 做封面標題與上架文案。

### 流程 C：Palmier 專案已經剪完但不好看

1. 用 `palmier-pro-shorts-finisher` 接續既有 `.palmier` 專案。
2. 加上封面區、HUD、低調水印、片尾動畫。
3. 如果 Palmier 沒刷新，另存新副本再打開。

## 學生安裝方式

下載這個 repo 後，把想用的 skill 內層英文資料夾複製到 `~/.claude/skills/`。

例如只安裝純訪談黑底字幕：

```bash
cp -r "skills/進階剪輯-純訪談黑底字幕-interview-reels-dialogue-editor/interview-reels-dialogue-editor" ~/.claude/skills/
```

一次安裝這四個進階剪輯/上架 skill：

```bash
cp -r "skills/進階剪輯-長片自動切短影音-video-shorts/video-shorts" ~/.claude/skills/
cp -r "skills/進階剪輯-Palmier短影音精修-palmier-pro-shorts-finisher/palmier-pro-shorts-finisher" ~/.claude/skills/
cp -r "skills/進階剪輯-純訪談黑底字幕-interview-reels-dialogue-editor/interview-reels-dialogue-editor" ~/.claude/skills/
cp -r "skills/影片上架包-rick-yt-publish/rick-yt-publish" ~/.claude/skills/
```

裝完後重開 Claude Code，再用自然語言描述需求即可。
