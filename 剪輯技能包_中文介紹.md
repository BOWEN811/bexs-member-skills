## 兩格判斷法

### 第 1 格：短影片專用版本

用 `ig-short-video-marketing-editor`。

只要目標是 IG Reels、Shorts、TikTok、直式短影片、9:16 行銷短片，就用這個。

它裡面有兩個風格分支：

- **粥粥訪談風**：純訪談、Podcast 感、黑底高級字幕、主持人和受訪者要分開。
- **直式拍螢幕預告風**：手機拍螢幕、手指指畫面、課程預告、會員公告、展示 Codex / PalmierPro / GitHub。

你可以這樣講：

```text
這支用短影片專用版本剪，打開專案給我看
```

```text
這支照粥粥訪談風做，Rick 跟受訪者不要混，字幕不能漏
```

```text
這支是直式拍螢幕預告，做成 IG Reels，字幕不要黑底
```

### 第 2 格：YouTube 長片專案精修版

用 `youtube-longform-project-finisher`。

只要目標是 YouTube 長片、課程長片、16:9 訪談、長教學、長片專案整理，就用這個。

它處理的是長片專案，不是 IG 短片。

你可以這樣講：

```text
這支 16:9 長片用 YouTube 長片專案精修版
```

```text
這支 YouTube 長片幫我整理字幕、音軌、章節，打開專案給我看
```

```text
這支課程長片幫我做專案精修，不要剪成短影音
```

## 強制規則：不能只跑 CLI 版本

這是 Rick 特別要求的剪輯標準。

不管用哪個剪輯 skill，都不能只在背景跑完，丟一個 mp4 檔案就結束。

每次都必須：

- 產生 `.palmier` 專案並打開 PalmierPro。
- 讓 Rick 在 PalmierPro timeline 上看得到 V 軌、A 軌、字幕素材/字幕軌與分段。
- A 軌要看得到 waveform。
- 字幕不能只燒進 mp4，必須有字幕素材、字幕段或 overlay 檔在軌道上。
- preview/review 頁只能當補充，不能取代 PalmierPro。
- 音軌不能中途沒聲音。
- 字幕不能漏，也不能跟聲音錯位。
- 如果是訪談，主持人和受訪者不能混在一起。
- 如果是短影片，畫面與字幕要放在 IG 安全區內。

簡單講：**要做出可以檢查的 PalmierPro 剪輯專案，不是只交一個黑箱成片。**

## 短影片專用版本細節

Skill：`ig-short-video-marketing-editor`

### 適合

- IG Reels
- YouTube Shorts
- TikTok
- 9:16 直式短影片
- 純訪談黑底字幕短片
- 手機直拍螢幕預告
- 課程公告
- 會員公告
- 自動剪輯短影片

### 兩個內部分支

| 分支 | 什麼時候用 | 重點 |
|---|---|---|
| 粥粥訪談風 | 純訪談、Podcast、沒有露臉 | 黑底、高級字幕、Rick/受訪者分色、逐字稿不能漏 |
| 直式拍螢幕預告風 | 手機拍螢幕、課程預告、展示工具 | 保留真實感、字幕不要黑底、上方標題放安全區 |

### 硬規則

- 9:16，1080x1920。
- 字幕繁體中文。
- 必須打開 PalmierPro 專案。
- 必須能看到音軌 waveform、字幕素材/字幕軌與分段。
- 字幕不能只燒進 mp4，必須在 timeline 上可檢查。
- 不可只跑 CLI / headless 版本。
- IG 安全區保守值：左 `92px`、右 `988px`、上 `170px`、下 `1640px`。

## YouTube 長片專案精修版細節

Skill：`youtube-longform-project-finisher`

### 適合

- YouTube 長片
- 16:9 課程影片
- 長訪談
- 長教學
- 長片字幕整理
- 長片音軌修整
- YouTube chapter / 章節
- 片頭、片尾、下三分之一、B-roll、長片包裝

### 不適合

- IG Reels
- Shorts
- TikTok
- 粥粥黑底訪談短片
- 手機拍螢幕預告

這些全部回到 `ig-short-video-marketing-editor`。

### 硬規則

- 16:9，不要錯裁成 9:16。
- 保留長片敘事，不要剪成短影音節奏。
- 必須打開 PalmierPro 專案。
- 必須能看到音軌 waveform、字幕素材/字幕軌、章節與片段切點。
- 字幕不能只燒進 mp4，必須在 timeline 上可檢查。
- 不可只跑 CLI / headless 版本。

## 影片上架包不是剪輯入口

`rick-yt-publish` 只用在影片剪完後。

它負責：

- YouTube 標題
- 影片簡介
- 章節文字
- hashtag
- 封面標題
- 封面提示詞

你可以這樣講：

```text
這支影片剪完了，給我標題、內文、封面提示詞
```

它不是剪輯工具，不要拿來剪片。

## 學生安裝方式

下載這個 repo 後，把想用的 skill 內層英文資料夾複製到 `~/.claude/skills/`。

只安裝短影片專用版本：

```bash
cp -r "skills/進階剪輯-IG短影片行銷自動剪輯-ig-short-video-marketing-editor/ig-short-video-marketing-editor" ~/.claude/skills/
```

只安裝 YouTube 長片專案精修版：

```bash
cp -r "skills/進階剪輯-YouTube長片專案精修-youtube-longform-project-finisher/youtube-longform-project-finisher" ~/.claude/skills/
```

全部安裝：

```bash
for d in skills/*/*/; do cp -r "$d" ~/.claude/skills/; done
```
