# 📊 表格分析 · data-analyzer

**這個幫你:丟一個 Excel/CSV,用講的就問出重點,還能畫圖。**

## 什麼時候用
有一份數據(銷售、成績、記帳、名單…)想快速看懂、或要回答「總共多少、誰最高、什麼趨勢」。

## 怎麼用
- 「分析這個 Excel,告訴我重點」
- 「這份銷售表,哪個月最高?幫我畫成長條圖」
- 「這個 CSV 幫我算各分類的總和」

## 我會做
1. 讀你的表格(用 python pandas,沒有我幫你裝)
2. 自動給總覽:有幾筆、哪些欄位、總和/平均/最大最小
3. 回答你的具體問題
4. 需要的話畫圖表(長條/折線/圓餅)存成圖檔給你

## 零依賴
全在你電腦上跑、資料不外流。
