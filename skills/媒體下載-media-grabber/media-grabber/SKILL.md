# ⬇️ 媒體下載 · media-grabber

**這個幫你:把 YouTube / TikTok / IG 的影片或音檔下載到電腦。**

## 什麼時候用
想離線看影片、留存教材、把影片轉成 mp3 音檔聽的時候。

## 怎麼用
- 貼一個影片連結 +「幫我下載」
- 「這支只要音檔,轉成 mp3」
- 「下載最高畫質」

## 我會做
1. 確認連結
2. 用 yt-dlp 下載(沒裝我幫你 `brew install yt-dlp`)
3. 影片存 mp4 / 音檔存 mp3,告訴你存在哪

## 依賴
需要 yt-dlp(免費開源、一行指令安裝)。**只供個人離線使用,請尊重版權。**
