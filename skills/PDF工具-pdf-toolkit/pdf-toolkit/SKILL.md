# 📄 PDF 工具 · pdf-toolkit

**這個幫你:在自己電腦上處理 PDF,不用裝軟體、不用把檔案傳到網路上。**

## 什麼時候用
要合併、拆開、抽頁、壓縮、或把 PDF 轉成文字的時候。

## 怎麼用
- 「把這兩個 PDF 合併」
- 「這個 PDF 太大,幫我壓縮」
- 「抽出這個 PDF 的第 3 到 5 頁」
- 「把這個 PDF 轉成文字」

## 我會做
1. 確認你的 PDF 檔在哪
2. 用本機工具(python 的 pypdf / qpdf,沒有我會幫你裝)處理
3. 結果存成新檔(不會蓋掉原檔)、告訴你存在哪

## 安全
全程在你電腦上跑、檔案不外流;原檔保留、只另存新檔。
