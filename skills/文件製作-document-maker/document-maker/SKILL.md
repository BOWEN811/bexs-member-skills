# 📑 文件製作 · document-maker

**這個幫你:用講的就做出 Word / Excel / PowerPoint / PDF,不用會 Office。**

## 什麼時候用
要交一份報告、做張試算表、生一份簡報、產一個 PDF 的時候。

## 怎麼用
- 「幫我把這些數字做成一份 Excel,要有合計公式」
- 「做一份 8 頁的產品介紹簡報」
- 「把這份內容做成 Word 報告,有標題跟分段」
- 「生成一份 PDF 收據」

## 我會做
1. 問清楚:做什麼類型、內容、要幾頁/幾欄
2. 用本機開源套件產出(Word=python-docx、Excel=openpyxl 含公式、PPT=python-pptx、PDF=reportlab;沒裝我幫你裝)
3. 存成檔案、告訴你存在哪、可直接打開交出去

## 零依賴
全在本機產檔,套件免費開源、自動安裝,檔案不外流。
