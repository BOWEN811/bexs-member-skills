# 📁 整理我的檔案 · file-organizer

**這個幫你:把亂七八糟的資料夾自動分類整齊,不刪檔、只歸位。**

## 什麼時候用
桌面或下載資料夾一堆檔案、找東西找半天的時候。

## 怎麼用(講人話就行)
- 「整理我的下載資料夾」
- 「桌面好亂,幫我分類」

## 我會做
1. 先看那個資料夾有什麼、列給你看(先不亂動)
2. 問你要「依類型分」還是「依日期分」
3. 建子資料夾(圖片 / 文件 / 影片 / 音樂 / 壓縮檔 / 其他),把檔案搬進去
4. **絕對不刪檔**,只搬位置;搬完給你「哪個檔搬去哪」的清單
5. 看不懂的檔先問你,不硬塞

## 安全
只搬不刪、動手前先給你看清單。
