# 📰 網頁摘要 · web-summarizer

**這個幫你:丟一個網址或長文,30 秒給你重點,不用看完。**

## 什麼時候用
看到一篇很長的文章/報導/部落格,沒時間讀完,想先抓重點。

## 怎麼用
- 貼一個網址 +「幫我看這篇」
- 「這篇文章太長,給我重點」

## 我會做
1. 抓那個網頁/文章的內容
2. 整理成:**一句話總結** → **3-5 個重點** → **關鍵數據或結論**
3. 你想深入哪一點,再追問

## 零依賴
用內建抓網頁能力,不用裝東西。
