# 🌐 提示詞翻譯 · prompt-translator

**這個幫你:打中文描述,出一段道地的英文生圖提示詞,直接貼進生圖工具用。**

## 什麼時候用
你要用 Midjourney / Stable Diffusion / ComfyUI 等英文生圖工具,但不想自己生英文 prompt。

## 怎麼用
- 「一個女生坐在咖啡廳看書,陽光從窗戶灑進來,電影感 → 翻成英文提示詞」

## 我會做
1. 讀懂你的中文描述
2. 轉成**道地、好用**的英文提示詞(自動補風格/光線/構圖/品質常用詞,不是死板直譯)
3. 附中文對照,讓你知道每段在講什麼

## 零依賴
純翻譯改寫,不用裝東西。
