PART 1每天都用得到(零依賴 · 新手友善)

📑
文件製作
document-maker
用講話的方式做出 Word / Excel / PowerPoint / PDF。Excel 會自動帶公式、PPT 會套版型,不用會 Office 操作。
怎麼用
「幫我把這三季數字做成 Excel,要有合計公式」/「做一份 8 頁產品簡報」
安裝
文件製作-document-maker
📊
表格分析
data-analyzer
丟一個 Excel / CSV,直接用問的:總和、平均、誰最高、什麼趨勢,還能畫圖表。不會樞紐分析、不會公式也能秒懂數據。
怎麼用
「分析這個 sales.csv,哪個月賣最好?畫成長條圖」
安裝
表格分析-data-analyzer
📁
檔案整理
file-organizer
把亂到爆的下載 / 桌面資料夾自動依類型或日期分類歸檔。只搬不刪、動手前先給你看清單。
怎麼用
「整理我的下載資料夾,依類型分」
安裝
檔案整理-file-organizer
📄
PDF 工具
pdf-toolkit
在你自己電腦上處理 PDF,不用裝軟體、不用上傳到可疑網站:合併、拆開、抽某幾頁、轉純文字、壓縮變小。原檔保留、只另存新檔。
怎麼用
「把這兩個 PDF 合併」/「這個 PDF 太大,壓縮一下」
安裝
PDF工具-pdf-toolkit
🖼️
圖片工具
image-toolkit
一次處理一堆圖:縮放尺寸、壓縮變小、轉格式(png/jpg/webp)、加文字浮水印。上傳前壓圖、整理相簿都用得到。原圖不動、結果另存。
怎麼用
「這個資料夾的圖全部縮到 1080 寬、轉成 jpg」
安裝
圖片工具-image-toolkit
📰
網頁摘要
web-summarizer
丟一個網址或長文,30 秒給你重點:一句話總結 → 3-5 個重點 → 關鍵數據。沒空看完長文時用。
怎麼用
貼網址 +「這篇幫我抓重點」
安裝
網頁摘要-web-summarizer
✍️
寫作助手
writing-assistant
寫 / 改 email、訊息、貼文、文章,語氣你選,還能「去 AI 味」—— 拔掉破折號、罐頭句、重複結構這些一看就是 AI 的痕跡。
怎麼用
「幫我寫一封跟客戶延期的道歉信,語氣誠懇」/「這段去掉 AI 味」
安裝
寫作助手-writing-assistant

PART 2創作 / 內容好用

⬇️
媒體下載
media-grabber
下載 YouTube / TikTok / Instagram / Facebook 的影片或音檔:可選畫質的 mp4、或只抓 mp3。離線看、留存教材、抽音檔都用得到。
怎麼用
貼連結 +「下載這支影片」/「只要 mp3 音檔」
安裝
媒體下載-media-grabber(需 yt-dlp,會幫你裝)
🌐
提示詞翻譯
prompt-translator
打中文描述 → 出道地英文生圖提示詞(餵 Midjourney / Stable Diffusion / ComfyUI)。自動補風格、光線、構圖常用詞,不是死板直譯。給不懂英文但要用英文生圖工具的人。
怎麼用
「一個女生在窗邊咖啡廳看書、陽光灑進來、電影感 → 翻成英文提示詞」
安裝
提示詞翻譯-prompt-translator

安裝怎麼裝(3 步驟,新手照做)

1. 下載這個包:GitHub 右上角綠色 Code → Download ZIP

2. 把技能複製到你的 Claude 技能資料夾(一次全裝):

for d in skills/*/*/; do cp -r "$d" ~/.claude/skills/; done

只想裝某一個:cp -r skills/檔案整理-file-organizer/file-organizer ~/.claude/skills/

3. 重開 Claude Code → 直接用講的就會觸發。

🎯 裝完怎麼用? 不用記任何指令! 你只要「用講的」,Claude 會自動挑出對的技能來做。你不用說「用某某技能」,只要說「翻成英文提示詞」「下載這支影片」,它看懂了就會自動用那個技能。就像跟一個很會做事的助理講話 —— 講得越清楚(給什麼、要什麼、多長),做得越準。

之後開放進階技能(要先裝環境)

這些超強但要先裝本地 AI 模型 / 轉錄工具,會另出進階包 + 安裝教學:🎥 YouTube 影片 → 中文逐字稿 + 重點報表　·　✂️ 長片 → 自動切多支短影音　·　🎙️ 聲音克隆　·　🔍 看圖反推提示詞。
