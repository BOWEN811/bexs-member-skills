# BEXS 會員技能包 · 給你的 AI 幫手裝上「超能力」

> 這裡每一個「技能」,就是幫你的 AI(Codex / Claude Code)多會一件事。

這一區只保留兩個正式剪輯入口，避免學生不知道該用哪一個。

> 新增 / 整理日期：**2026-06-27**

完整中文介紹看這份：

| 新增日期 | 你要做的事 | 要用的 skill | 怎麼開口 |
|---|---|---|---|
| 2026-06-27 | IG / Reels / Shorts 短影音行銷自動剪輯 | **ig-short-video-marketing-editor** | 「這支用短影片專用版本剪，打開專案給我看」 |
| 2026-06-27 | YouTube / 課程 / 訪談 16:9 長片專案精修 | **youtube-longform-project-finisher** | 「這支 16:9 長片用 YouTube 長片專案精修版」 |

> 💡 **剪輯只分兩格**：9:16 短影音都用 `ig-short-video-marketing-editor`；16:9 YouTube 長片都用 `youtube-longform-project-finisher`。
>
> ✅ **新版剪輯共同標準**：剪之前先抽完整字幕/逐字稿並理解全片；不得用全量 PNG/OCR 讀字幕，PNG 只做少量視覺抽查；正式字幕一律白色文字 + 黑色描邊，且省 PNG 不代表跳過字幕樣式驗收；剪完後除了打開 PalmierPro，還要回覆「已完成：剪輯完成。」、附 3-5 條可直接複製的生成提示詞，並全自動接上架包。
>
> 🚀 `rick-yt-publish` 是影片上架文字輔助，不是剪輯入口；兩個剪輯 skill 剪完後會自動接它產出標題、簡介、章節、hashtag、封面提示詞。CLI / Claude Code / Codex 會直接接本地 skill；Claude App / GPT App 會在最後回覆直接輸出同等上架包。

## 📥 怎麼安裝(3 步驟,新手照做)

1. **下載這個包**:右上角綠色 `Code` → `Download ZIP`(或會用 git 的 `git clone`)
2. **把技能複製到你的 AI agent 技能資料夾**:
   ```bash
   # Claude Code：一次全部裝(複製每個技能的內層英文資料夾)
   for d in skills/*/*/; do cp -r "$d" ~/.claude/skills/; done
   ```
   只想裝某一個:`cp -r skills/檔案整理-file-organizer/file-organizer ~/.claude/skills/`

   Codex 也能用同一份 `SKILL.md`；把內層英文技能資料夾放到 Codex 可讀取的 skills 資料夾，例如 `~/.codex/skills/`，或使用課堂提供的安裝方式。
3. **重開 Claude Code / Codex** → 直接用講的就會觸發(例如「整理我的下載資料夾」)

> 💡 資料夾用「**中文名-英文名**」包著方便你認;裡面那層**英文資料夾**才是真正的技能包。

## 🎯 裝完之後怎麼用?(超重要)

**不用記任何指令!** 裝好重開 Claude Code / Codex 後,你只要**用講的**,AI 會自動挑出對的技能來做。下面每個技能配一句**範例**,照著講就會動:

| 技能 | 你直接這樣打 |
|---|---|
| 📑 文件製作 | 「幫我把這三季的數字做成 Excel,要有合計公式」/「做一份 8 頁產品簡報」 |
| 📊 表格分析 | 「分析這個 sales.csv,哪個月賣最好?畫成長條圖」 |
| 📁 檔案整理 | 「整理我的下載資料夾,依類型分」 |
| 📄 PDF 工具 | 「把這兩個 PDF 合併成一個」/「這個 PDF 太大,壓縮一下」 |
| 🖼️ 圖片工具 | 「這個資料夾的圖全部縮到 1080 寬、轉成 jpg」 |
| 📰 網頁摘要 | 貼網址 +「這篇幫我抓重點」 |
| ✍️ 寫作助手 | 「幫我寫一封跟客戶延期的道歉信,語氣誠懇」/「這段去掉 AI 味」 |
| ⬇️ 媒體下載 | 「下載這支 https://youtube.com/watch?v=…」/「這支只要 mp3 音檔」 |
| 🌐 提示詞翻譯 | 「一個女生在窗邊咖啡廳看書、陽光灑進來、電影感 → 翻成英文提示詞」 |
| 🎨 視覺設計 | 「幫我做一張春節促銷海報,喜慶風」 |
| 🧩 UIUX 設計庫 | 「我要做一個賣課程的登陸頁,給我配色+版型建議」 |
| 🏷️ 品牌規範 | 「把這份簡報套上我的品牌色」 |
| ✂️ IG 短影片行銷自動剪輯 | 「這支用短影片專用版本剪，先抽完整字幕理解全片，字幕白字黑描邊，打開專案給我看，剪完自動接上架包」 |
| 🎬 YouTube 長片專案精修 | 「這支 16:9 長片用 YouTube 長片專案精修版，先完整抽字幕理解全片，剪完自動接 rick-yt-publish 上架包」 |
| 🚀 影片上架包 | 「這支影片已剪完，給我標題、內文、章節、hashtag、封面提示詞」 |

> 💡 **訣竅**:就像跟一個很會做事的助理講話 —— 你說「我要什麼」,它就去做。講得越清楚(給什麼、要什麼、多長),做得越準。
>
> 🔁 **它會自己挑技能**:你不用說「用提示詞翻譯技能」,只要說「翻成英文提示詞」,它看懂了就會自動用那個技能。
